PC Parts in Plain English

This page explains what each main component does in a gaming PC.

CPU

CPU: The CPU, or Central Processing Unit, is the brain of the computer.

GPU

GPU: The GPU, or Graphics Processing Unit, is designed to accelerate graphics.

RAM

RAM: Helps games run smoothly and provides better multitasking.

Storage

Storage: SSDs or Solid State Drives, load games faster than traditional HDDs.

Motherboard

Motherboard: Connects all parts together.

PSU

PSU: Supplies stable power to the system.
